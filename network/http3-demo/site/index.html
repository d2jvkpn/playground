This server supports HTTP/3, but browsers often load the first page over HTTP/2, learn the Alt-Svc advertisement, and only then try HTTP/3 on a later connection. If you see HTTP/2.0 here, it means this specific browser request used HTTP/2, not that the backend lacks HTTP/3.
