# TUI / CLI UI**

## 1. Box Drawing 边框线

```text
─ │ ┌ ┐ └ ┘ ├ ┤ ┬ ┴ ┼
═ ║ ╔ ╗ ╚ ╝ ╠ ╣ ╦ ╩ ╬
╭ ╮ ╰ ╯
╱ ╲ ╳
```

常用组合：

```text
┌────────────┐
│  Content   │
└────────────┘

╭────────────╮
│  Content   │
╰────────────╯
```

## 2. 列表 / 树结构

```text
• ◦ ▪ ▫
- *
├──
└──
│
┬ └ ├
```

示例：

```text
project
├── src
│   ├── main.rs
│   └── ui.rs
└── Cargo.toml
```

## 3. 状态符号

```text
✓ ✔ ✅  成功
✗ ✘ ❌  失败
! ⚠ ▲  警告
i ℹ     信息
? ❓    疑问
```

更适合纯 TUI 的非 emoji 版本：

```text
✓ ✗ ! i ?
```

## 4. 箭头 / 导航

```text
← → ↑ ↓
↵ ↩
⇧ ⇩
➜ ➔ ➤
› ‹ » «
```

常用：

```text
> selected item
➜ current step
› submenu
```

## 5. 进度条 / Block

```text
█ ▉ ▊ ▋ ▌ ▍ ▎ ▏
▓ ▒ ░
■ □
▪ ▫
▬
```

示例：

```text
████████░░ 80%
▓▓▓▓▓▒▒▒▒ 50%
```

## 6. Spinner / Loading

```text
| / - \
◐ ◓ ◑ ◒
⠋ ⠙ ⠹ ⠸ ⠼ ⠴ ⠦ ⠧ ⠇ ⠏
⠁ ⠂ ⠄ ⡀ ⢀ ⠠ ⠐ ⠈
```

经典：

```text
⠋ Loading
⠙ Loading
⠹ Loading
```

## 7. Checkbox / Radio

```text
[ ] [x] [-]
☐ ☑ ☒
○ ●
◯ ◉
```

示例：

```text
[ ] Enable logs
[x] Use cache

○ Option A
● Option B
```

## 8. 分隔符 / 装饰

```text
─ ━ ═
· •
◆ ◇
◈
✦ ✧
```

示例：

```text
──────── Settings ────────
```

## 9. 表格相关

```text
│ ┃
─ ━
┌ ┬ ┐
├ ┼ ┤
└ ┴ ┘
```

示例：

```text
┌──────┬────────┐
│ Name │ Status │
├──────┼────────┤
│ API  │ OK     │
└──────┴────────┘
```

## 10. ASCII fallback

当终端字体或编码不稳定时，用这些最安全：

```text
+ - | /
\ *
> <
[ ] ( )
x
!
?
=
#
```

示例：

```text
+------------+
|  Content   |
+------------+
```

## 11. Nerd Font / 图标类，可选

需要安装 Nerd Font 才稳定显示：

```text
   
   
   
  
```

这类适合做状态栏、文件树、Git 状态，但不建议作为默认，因为用户终端未必支持。

## 推荐基础字符集

如果想做一个兼容性较好的 TUI，我建议先用这一组：

```text
─ │ ┌ ┐ └ ┘ ├ ┤ ┬ ┴ ┼
╭ ╮ ╰ ╯
├── └──
✓ ✗ ! i ?
← → ↑ ↓
█ ░
[ ] [x]
○ ●
›
```
